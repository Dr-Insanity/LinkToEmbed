DisGhost's candidate for detecting VPN users.

Press start to check whether you're using a VPN or not.

But before that, there's some things you need to know

  • The VPN checker uses your webbrowser's feature WebRTC, which needs to be enabled.
  • WebRTC is both used by attackers and normal developers alike. It's not a bad feature, but unfortunately just
    unethically used. Most webbrowsers have this for that reason disabled by default (For security reasons).
